# UART Transmitter and Receiver using Verilog

## Overview
Designed and implemented UART Transmitter and Receiver modules for serial communication using Verilog HDL.

## Features
- 8-bit data transmission
- Start bit detection
- Stop bit generation
- Serial-to-parallel conversion
- FSM-based control logic
- Functional verification using ModelSim

## Tools Used
- Verilog HDL
- ModelSim
- VS Code

## Project Structure
rtl/
tb/

## Simulation
Waveforms were verified using ModelSim to validate correct data transmission and reception.
